# Cruise-control installer

Automagical cross-platform *nix package management script. Automatically install/remove a list of packages on various *nix distros. Can also handle a single package via arguments. Also supports a user-specified package management command for extra portability.

Supports aptitude, apt-get, yum, zypper, pacman, emerge, pkg (SunOS and BSD), and pkg_add/pkg_delete package managers.
Tested on Arch, CentOS, Debian, FreeBSD, Gentoo, Solaris 11, OpenBSD, OpenSUSE, Ubuntu.
